Fill Missing Days

This tool will analyze employee attendance records and add entries for missing days:

  • Missing weekdays will be marked as No Work
  • Missing weekend days will be marked as Weekend
Leave blank to process all employees
If no date range is specified, the system will use each employee's first and last attendance record.
How It Works

This tool helps ensure your attendance data is complete by:

  1. Finding date ranges for each employee (first attendance to last attendance)
  2. Identifying days within that range with no attendance records
  3. Adding "No Work" entries for missing weekdays
  4. Adding "Weekend" entries for missing weekend days
Important: This process may add many records to your database. Consider creating a backup first.
